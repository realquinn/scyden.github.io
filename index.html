Installing the Scyden addon

Current Repository Version: 1.0.0

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://scyden.github.io/
  • Name it "Scyden Repo" without the quotes
  • Head to " My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select "Scyden Repo", and install repository.scyden.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the "Scyden Repo" menu item
  • Open "Video add-ons" and install Scyden
  • (Optional) It's suggested to also install the context menu addon

Enjoy!
